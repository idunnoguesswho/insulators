Rectangle to Rectangle

Dims Required: Large and small dims, actual

or slant height.

1. Draw the working drawing (end view) include

the cross hairs, divide & label (best to

include the break lines).

2. Draw the true length triangle and develop the

true lengths.

3. Use the dimensions on the working drawing

and true length triangle to develop the

stretch-out using the triangulation

method.
