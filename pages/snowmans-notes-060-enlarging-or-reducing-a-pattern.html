Enlarging or Reducing a

Pattern

Example: To increase the size of this pattern

to 3 times the original size, the corner 1-2-3

Is a right angle to 1’-2’-3’ also a right angle

can therefor be made 3 times as large (in

this case).

Draw construction lines for any arbitrary

point O to the edge of the original pattern

and add 2 more times the outside the

original. This action increases the pattern

size to 3 times the original. Other enlarging

or reducing can be done the same way.
