C Bevels: Square Method

1. Draw a 90° angle large enough to fit the

OD of the pipe covering. (Legs)

2. Mark pipe radius on both legs.

3. Mark Insulation radius on both legs.

Connect matching radii. Extend line

for insulation radii, outward to

allow for crimp.

4. Set dividers from insulation thickness ->

crimp allowance. Then scribe a

circle.
