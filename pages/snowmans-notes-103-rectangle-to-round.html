Rectangle to Round

Dims Required: Dimensions of rectangle,

OD of round, actual or slant height.

1. Draw the working drawing (end view). Start

with cross hairs the circle then

rectangle. Label divisions and connect to

corners for break lines.

2. Draw the true length triangle and develop the

true lengths using the working drawing.

3. Develop the stretch-out using dimensions on

the working drawing and true length

triangle.
